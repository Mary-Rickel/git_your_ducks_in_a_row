# 🦆 Ducks in a Row

> Code repository for PDP 2026 Team 12 Activity

A playful data science project that demonstrates image manipulation and visualization techniques using duck images and scipy transformations.

## 📋 Overview

This project showcases various image processing techniques including rotation, shifting, and manipulation of duck images using Python's scientific computing libraries. It's designed as an educational activity for Team 12 in the Professional Development Program 2026.

## 🎯 Project Contents

- **`plot.py`** - Main image manipulation script that applies transformations to duck images
- **`put_in_row.ipynb`** - Jupyter notebook containing the interactive analysis and results
- **`image1.png` through `image5.png`** - Duck images used for demonstration
- **`Security.md`** - Security policy and vulnerability reporting guidelines

## 🔧 Technologies Used

- **Python 3** - Core programming language
- **Matplotlib** - Image display and visualization
- **NumPy** - Numerical array operations
- **SciPy** - Advanced scientific computing (image rotation, shifting)

## 🚀 Getting Started

### Prerequisites

Make sure you have Python installed with the required packages:

```bash
pip install matplotlib numpy scipy
```

### Running the Project

1. **Using the Python script:**
   ```bash
   python plot.py
   ```

2. **Using the Jupyter notebook:**
   ```bash
   jupyter notebook put_in_row.ipynb
   ```

## 📝 What It Does

The project demonstrates several image manipulation techniques:

- **Image rotation** - Rotating duck images by specific angles
- **Image shifting** - Rolling images along axes to reposition them
- **Image flipping** - Flipping images vertically and horizontally
- **Brightness adjustment** - Applying mathematical transformations to modify image intensity

All transformations are applied to duck images and visualized in a 4-panel subplot layout.
